# Bowl

Simulation of bowling like games


## Requirements

For web:
- Python >= 3.8
- Flask == 2.0.1

```shell
$ pip install -r requirements.txt
```
For CLI:
- Python >= 3.8


## Run

For web:
```shell
$ python src/web.py
```
Open your browser and enter [http://127.0.0.1:5000/]()

For CLI:
```shell
$ python src/main.py
```


## Testing

Install dev dependency

```shell
$ pip install -r requirements-dev.txt
```
And run

```shell
$ pytest -v
```
